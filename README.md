# Face Mask Detection 😷

## 📌 Project Overview
This project detects whether a person is wearing a mask or not using Deep Learning.

## 📂 Dataset

Dataset used in this project can be downloaded from:
https://www.kaggle.com/datasets/prithwirajmitra/covid-face-mask-detection-dataset

After downloading, arrange like this:

dataset/
 ├── with_mask/
 └── without_mask/

## 🛠 Technologies Used
- Python
- TensorFlow / Keras
- OpenCV

## 🚀 Features
- Real-time face mask detection
- Webcam integration
- CNN-based classification

## ▶️ How to Run
1. Run train.py
2. Run detect.py

## 📊 Output
- Displays Mask / No Mask in real-time
